# graphql-devc

GraphQL Server Demo for Facebook DevC Malang April Event

## Requirement

* node LTS
* docker
* docker-compose

## Run This Project

* `npm install`
* `docker-compose up -d`
* create database `warcraft` in mongo
* `npm run seed`
* `npm start`
* open `localhost:3000/graphiql` for graphiql

## Test

`npm test`
